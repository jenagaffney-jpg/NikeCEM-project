# Nike Customer Enablement Demo

This project demonstrates Code ↔ Canvas ↔ Code workflows using:

- Figma Dev Mode
- Figma MCP Server
- Cursor
- Claude Code

## Objective

Show how enterprise teams can accelerate product development by connecting design and development workflows through AI-assisted tooling.

## Demo Workflow

1. Create code in Cursor
2. Connect Figma via MCP
3. Generate Canvas from code
4. Make design changes
5. Roundtrip updates back to code
6. Collaborate and ship faster
